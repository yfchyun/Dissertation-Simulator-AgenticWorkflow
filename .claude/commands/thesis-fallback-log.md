# Thesis Fallback Log

Display fallback events from the thesis workflow.

## Protocol
1. Read SOT fallback_history array
2. Read files in fallback-logs/ directory
3. Display each event: timestamp, tier switch (1→2 or 2→3), reason, affected step
4. Show quality impact assessment
