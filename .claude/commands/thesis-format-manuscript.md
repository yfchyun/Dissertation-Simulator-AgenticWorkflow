# Format Manuscript

Convert thesis to journal submission format.

## Protocol

1. Read thesis draft and target journal from SOT
2. Delegate to @manuscript-formatter:
   - Apply target journal's style guide
   - Reformat citations and references
   - Adjust heading levels and structure
   - Format tables and figures per journal requirements
   - Prepare cover letter template
   - Check word/page limits
3. Output formatted manuscript to `thesis-output/{project}/submission/`
4. Generate submission checklist with journal-specific requirements
