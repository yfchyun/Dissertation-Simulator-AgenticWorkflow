# Learning Progress

Display comprehensive learning progress report.

## Protocol

1. Read SOT `learning_progress` field
2. Display progress table:
   - Track completion percentage per track (1-8)
   - Quiz scores and dates
   - Practice exercise completion status
   - Overall learning readiness assessment
3. Recommend next steps based on gaps
4. If all tracks complete: suggest transitioning to research mode
